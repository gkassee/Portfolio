NEIGHBORINO

A message board designed to bring neighborhoods together, Neighborino allows users to communicate with members within a 1 mile radius of their location.

Ruby on Rails Twitter API OAuth Google Maps API W3C Geoloacation PostgreSQL CRUD
Using OAuth through Twitter, users can easily create an account. Once a user inputs their home address, they will receive Tweet notifications whenever a post is made in their neighborhood.

BANDLY TREE

A single page application that allows users to explore the musical influences of their favorite musicians.

Ruby on Rails JavaScript jQuery AJAX Spotify API PostgreSQL
Bandly Tree was conceived and built to a functional state by a team of 4 during a one week sprint. The Nokogiri Ruby Gem is used to scrape musician data from AllMusic.com. The Spotify API is included to allow users to listen to samples of a musician's top songs.


*This project is in ongoing development.
