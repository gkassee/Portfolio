    NEIGHBORINO

    *This project was my final project at the New York Code + Design Academy.

    View

    A message board designed to bring neighborhoods together, Neighborino allows users to communicate with members within a 1 mile radius of their location.

    Using OAuth through Twitter, users can easily create an account. Once a user enters their home address, they will receive Tweets notifications whenever a post is made in their neighborhood.

    Technology: Ruby on Rails, Twitter API, OAuth, Google Maps API W3C Geolocation, HTML, CSS
